(static) sumfp(first, second) → {number}

A function to add two Fibonacci primes.

Sums two numbers only if both are Fibonacci primes. Otherwise the function returns -1 to indicate an error.
A Fibonacci prime is a Fibonacci number that is prime.
Parameters:

first number The first number to add.
